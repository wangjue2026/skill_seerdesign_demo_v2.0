# L 型导航页面通用内容模块规范

## 使用声明

本规范只定义 L 型导航页面中的页面模板结构、视觉规格、间距规则、交互行为和验收标准。

本规范不提供任何具体框架代码实现。

生成代码时，必须以当前项目的技术栈 Skill 为准。

如果本规范与项目技术栈约束发生冲突，优先遵守项目技术栈约束。

禁止在本规范中直接使用以下内容：

- React 代码
- JSX / TSX
- TypeScript 示例
- import 语句
- useState / useEffect
- className
- antd
- @ant-design/icons
- XxxOutlined 图标组件
- Vue3 Composition API
- 具体组件库内部 class 选择器

本规范中的 Table、Button、Input、Tree、Pagination、Toolbar 等名称仅表示设计语义，不代表必须使用某个框架或组件库的同名组件。

---

## 一、整体结构

L 型导航页面的内容区由一个外层容器包裹，内部为内容卡片。

内容卡片内部包含：

1. 页面标题/页头
2. 业务内容区（操作与筛选区、表格、分页等）

### 布局层级

```
┌─────────────────────────────────────────────────────────────────┐
│ 外层容器 (背景色: #F7F8FA, padding: 12px 或无)                   │
│ ┌─────────────────────────────────────────────────────────────┐ │
│ │ 内容卡片 (白色, 圆角 2px)                                   │ │
│ │ ┌─────────────────────────────────────────────────────────┐ │ │
│ │ │ A. 页面标题/页头 (Page Title) - 48px                     │ │ │
│ │ └─────────────────────────────────────────────────────────┘ │ │
│ │                     ↓ 间距 8px                              │ │
│ │ ┌─────────────────────────────────────────────────────────┐ │ │
│ │ │ B. 操作与筛选区 (Toolbar)                                │ │ │
│ │ └─────────────────────────────────────────────────────────┘ │ │
│ │                     ↓ 间距 8px                              │ │
│ │ ┌─────────────────────────────────────────────────────────┐ │ │
│ │ │ C. 数据展示区 (Table)                                    │ │ │
│ │ └─────────────────────────────────────────────────────────┘ │ │
│ │                     ↓                                       │ │
│ │ ┌─────────────────────────────────────────────────────────┐ │ │
│ │ │ D. 分页区 (Pagination) - 40px                           │ │ │
│ │ └─────────────────────────────────────────────────────────┘ │ │
│ └─────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
```

### 垂直堆叠顺序

| 区域 | 说明 |
|-----|------|
| A. 页面标题/页头 (Page Title) | 位于内容卡片最上方，全宽 |
| B. 操作与筛选区 (Toolbar) | 页面标题/页头下方，与页面标题/页头间距 8px |
| C. 数据展示区 (Table) | 操作与筛选区下方，与工具栏间距 8px |
| D. 分页区 (Pagination) | 位于容器底部 |

## 三、页面标题/页头规范

### 3.1 基础样式

页面标题/页头高度固定为 48px。

页面标题/页头必须位于内容卡片顶部。

### 3.2 左边距

- 页面标题/页头的左边距为 16px（相对于内容卡片）
- 内容卡片拥有左右 12px 内间距，因此标题文字实际左边距为 28px（12px + 16px）

### 3.3 组件使用要求

页面标题/页头规范详见 [comp-page-title.md](comp-page-title.md)（基础规范、基础页面标题/页头、下钻页页面标题/页头、分层 tabs 页标题）

如果项目中已存在统一的页面标题栏组件，应优先使用该组件。

返回图标与标题文字间距为 8px。

下钻页返回图标使用向左箭头图标。

## 四、页面内容卡片规范

### 4.1 基础样式

| 属性 | 规范值 |
|------|--------|
| 背景色 | 纯白 (#FFFFFF) |
| 圆角 | 2px |
| 边框 | 无 |

### 4.2 内间距

| 属性 | 规范值 |
|------|--------|
| 左右内间距 | 12px（padding: 0 12px） |
| 上下内间距 | 无 |

不得使用组件库默认卡片内边距覆盖该规范。

禁止出现上下 16px、左右 20px 等默认卡片内间距。

### 4.3 内容放置规则

所有业务内容必须放置在内容卡片内。

业务内容包括但不限于：

- 筛选区
- 操作按钮区
- 表格
- 表单
- 图表
- 详情信息
- 分页

内容卡片内部不允许再次出现页面标题/页头，也不允许重复展示当前页面标题。

### 4.4 业务内容与卡片边缘的关系

内容卡片内部的业务内容不得通栏贴到卡片左右边缘。

业务内容的左侧视觉边缘必须距离内容卡片左边缘 12px。

业务内容的右侧视觉边缘必须距离内容卡片右边缘 12px。

例如表格放入内容卡片时，表格不得直接贴齐卡片左右边缘，必须受到卡片左右 12px 内间距约束。

### 4.5 卡片内边距唯一来源原则

内容卡片已经统一定义左右内间距为 12px、上下内间距为 0px。

因此，内容卡片内部的一级业务模块不再单独设置左右 padding、左右 margin 或额外容器间距。

适用模块包括但不限于：

- Toolbar 操作筛选区
- Table 表格区
- Pagination 分页区
- Form 表单区
- Chart 图表区

这些模块在水平方向上应充满内容卡片的可用内容区，而不是充满卡片外边界。

视觉结果应表现为：

- 按钮左边缘距离内容卡片左边缘 12px；
- 表格左边缘距离内容卡片左边缘 12px；
- 分页器右边缘距离内容卡片右边缘 12px；
- 这些 12px 间距只能来自内容卡片自身的左右内间距；
- 不允许在 Toolbar、Table、Pagination 等子模块上重复设置左右 padding 或 margin。

禁止出现以下情况：

1. 内容卡片已有左右 12px 内间距，Toolbar 又额外设置左右 12px padding。
2. 内容卡片已有左右 12px 内间距，Table 又额外设置左右 12px padding。
3. 内容卡片已有左右 12px 内间距，Pagination 又额外设置左右 12px padding。
4. 子模块因重复设置间距，导致视觉间距变成 24px 或更大。
5. 为了让表格贴边而取消内容卡片的统一左右内间距。

## 五、布局关系总结

- 内容卡片位于外层容器内部。
- 基础表格页：外层容器四周 padding 为 12px，内容卡片与容器各边间距均由容器 padding 控制。
- 左树表格页：外层容器无 padding，内容卡片铺满容器。
- 内容卡片内部：页面标题/页头、操作与筛选区、表格、分页区上下排列，间距统一为 8px。
- 内容卡片内部业务内容左右间距为 12px，由卡片自身内间距提供。

## 六、设计原则

| 原则 | 说明 |
|------|------|
| 全屏原则 | 内容卡片高度占满屏幕，基础表格页仅保留外层容器 padding |
| 统一间距原则 | 所有区域间距统一为 8px |
| 最小间距原则 | 按钮组内部间距 8px，按钮与搜索框间距 8px |
| 表格无边框原则 | 表格外层无边框线 |
| 分页固定原则 | 分页区吸附在内容卡片底部 |
| 组件复用原则 | 页面标题/页头、操作与筛选区、数据展示区、分页区在不同页面类型间保持一致 |

---

## 七、禁止项

禁止出现以下实现：

1. 页面标题/页头未使用统一组件，而是临时手写标题文字。
2. 页面标题/页头为空，或标题显示为路由路径、英文 key、占位文字。
3. 内容卡片内部出现重复页面标题。
4. 内容卡片内部业务内容通栏贴到卡片左右边缘。
5. 内容卡片使用组件库默认内间距，例如上下 16px、左右 20px。
6. Toolbar、Table、Pagination 等子模块重复设置左右 padding 或 margin。
7. 在表格外层添加边框线。
8. 分页区与表格区之间出现额外间距。
9. 改变页面标题/页头高度。
10. 所有组件库默认样式覆盖未限制在当前页面模板作用域内，污染全局组件样式。

---

## 八、验收标准

页面实现完成后，必须满足以下标准：

1. 外层容器背景色为 #F7F8FA。
2. 内容卡片背景为白色，圆角为 2px，无边框。
3. 内容卡片左右内间距为 12px，上下内间距为 0px。
4. 页面标题/页头高度固定为 48px。
5. 页面标题/页头左边距为 16px（相对于内容卡片）。
6. Toolbar 与页面标题/页头间距为 8px。
7. Toolbar 与表格区间距为 8px。
8. 表格区域无外边框线。
9. 分页区高度固定为 40px，顶部有分割线。
10. 刷新按钮必须为 32x32px 外框，带边框线。
11. 内容卡片内部所有一级业务模块的视觉左右边缘与卡片内容区边界对齐。
12. 业务内容距离内容卡片左右边缘均为 12px，且该间距只由内容卡片提供。

#### 外层容器

| 属性 | 规范值 |
|------|--------|
| 背景色 | var(--color-graphite-l40) / #F7F8FA |
| padding | 12px（四周） |

#### 操作与筛选区 (Toolbar)

**间距规范**：

| 位置 | 间距值 |
|-----|--------|
| 与页面标题/页头间距 | 8px |
| 与数据展示区间距 | 8px |
| 左右内间距 | 0（继承内容卡片的 12px 内间距） |

**结构规则**：

Toolbar 左右分栏：
- 左侧为操作按钮组
- 右侧为筛选与快捷操作组

**左侧按钮组 (Action Group)**：
- 第一个按钮为主按钮（Primary），用于页面主操作，例如"新建"
- 后续按钮为普通按钮（Default），例如"导出""批量操作"
- 按钮高度统一为 32px
- 按钮之间水平间距为 8px
- 按钮图标为可选项，由当前组件库映射

**右侧筛选组 (Filter Group)**：
- 搜索框宽度为 240px
- 刷新按钮尺寸为 32px × 32px
- 刷新按钮需要有边框（边框颜色 graphite-l30, #E1E5EB）
- 筛选项之间水平间距为 8px

**验收标准**：
- 左侧按钮组靠左对齐
- 右侧筛选组靠右对齐
- 左右区域垂直居中
- Toolbar 与页面标题/页头间距为 8px
- Toolbar 与表格区间距为 8px

#### 表格区 (Table)

**结构规则**：

| 属性 | 规范值 |
|-----|--------|
| 与 Toolbar 间距 | 8px |
| 外边框线 | 无 |
| 左右内间距 | 0（继承内容卡片的 12px 内间距） |
| 横向滚动 | 内容展示不完整时出现横向滚动条，不要拉高表格高度 |
| 操作列冻结 | 有横向滚动条时，操作列冻结在右侧 |

**验收标准**：
- 表格区域无外边框线
- 表格高度固定，内容超出时横向滚动
- 有横向滚动时操作列固定可见

#### 分页区 (Pagination)

**结构规则**：

| 属性 | 规范值 |
|-----|--------|
| 位置 | 吸附在内容卡片底部 |
| 左右内间距 | 0（继承内容卡片的 12px 内间距） |
| 高度 | 40px |
| 顶部分割线 | 1px, graphite-l30 / #E1E5EB |
| 对齐方式 | 靠右对齐 |

**验收标准**：
- 分页区高度固定为 40px
- 顶部有 1px 分割线
- 分页控件靠右对齐

页面标题/页头规范与基础表格页完全一致，支持三种类型：基础页面标题/页头、下钻页页面标题/页头、分层 tabs 页页面标题/页头。

页面标题/页头规范详见 [comp-page-title.md](comp-page-title.md)

#### 左侧内容区 (Left Content Area)

左侧内容区用于放置树形组件（Tree）或纵向菜单区域，宽度固定为 240px。

**尺寸规范**：

| 属性 | 规范值 |
|-----|--------|
| 宽度 | 240px（固定，不可伸缩） |
| 最小宽度 | 240px |
| 最大宽度 | 240px |
| 与页面标题/页头间距 | 0（无间距，直接跟在页面标题/页头下方） |
| 与右侧区域间距 | 12px |
| 右边框 | 1px，分割线颜色 (graphite-l30, #E1E5EB) |
| 上下内间距 | 12px（内容卡片内间距） |

**左树组件规范**：

左树的样式以 [comp-tree.md](comp-tree.md) 中定义的 Tree 组件规范为准。

Tree 组件核心特性：
- 节点高度：32px
- 选中态：蓝色渐变背景 + 3px 左侧激活条
- 图标：12x12px，颜色 graphite-d20
- 文字：12px/400，颜色 graphite-d40
- 层级缩进：每级 12px
- 搜索框：可选，高度 32px

**左树选择交互**：

状态定义：系统需要维护当前选中的树节点集合。

触发条件：当用户点击左侧树节点时，触发节点选择逻辑。

状态变化：
- 更新当前选中的树节点
- 右侧表格根据选中的树节点刷新数据
- 如果树节点无数据，右侧表格显示空状态
- 选中态必须清晰可见

验收标准：
- 点击左侧树节点后，节点进入选中态
- 右侧表格内容跟随左树选择变化
- 左侧树区域宽度保持 240px
- 右侧内容区不被树区域挤压变形

#### 右侧内容区 (Right Content Area)

右侧内容区包含操作与筛选区、数据展示区、分页区，定义与基础表格页完全一致。

**布局规范**：

| 属性 | 规范值 |
|-----|--------|
| 宽度 | 自动填充剩余空间 |
| 最小宽度 | 600px（避免过窄） |
| 与左侧内容区间距 | 12px |
| 上下内间距 | 12px（内容卡片内间距） |
| 内部区域间距 | 8px（操作与筛选区、数据展示区、分页区，以及后续新增内容） |

验收标准：
- 右侧内容区自动填充剩余空间
- 右侧内容区最小宽度不低于 600px
- 右侧内容区与左侧内容区间距为 12px
- 右侧内部各区域之间间距为 8px

#### 左树表格页设计原则

| 原则 | 说明 |
|------|------|
| 固定宽度原则 | 左侧内容区宽度固定为 240px，不可伸缩 |
| 响应式原则 | 右侧内容区自动填充剩余空间 |
| 对齐一致原则 | 左侧内容区和右侧内容区都继承内容卡片的 12px 左右间距 |
| 组件复用原则 | 页面标题/页头、操作与筛选区、数据展示区、分页区与基础表格页保持一致 |
| Tree 规范原则 | 左树组件严格遵循 Tree 组件规范 |

#### 左树表格页验收标准

- 左侧内容区宽度固定为 240px，不可调整
- 左树组件样式遵循 Tree 组件规范
- 右侧内容区最小宽度设置为 600px，避免过窄
- 左侧内容区和右侧内容区之间有 12px 间距
- 左侧内容区右侧有 1px 分割线
- 页面标题/页头全宽，横跨左右两个区域
- 点击左侧树节点后右侧表格内容跟随变化
- 禁止左侧内容区宽度变化
- 禁止右侧内容区宽度小于 600px

#### 外层容器

与基础表格页一致，padding 为 12px 四周。

#### 页面标题/页头

- 高度 48px，与基础表格页完全一致，详见 [comp-page-title.md](comp-page-title.md)
- 页面标题/页头的左边距为 0px
- 内容卡片拥有左右 12px 内间距，因此标题文字实际左边距为 12px（由卡片内间距提供，标题自身不设置额外左间距）

#### 表单配置内容区 (Form Content)

| 属性 | 规范值 |
|------|--------|
| 左右内间距 | 20px（内容卡片自身 12px + 表单区 20px = 视觉总间距 32px） |
| 上下内间距 | 不额外设置，由内容卡片控制 |
| 表单规范 | 遵循 [design-form.md](design-form.md) 表单规范 |
| 滚动 | 内容超出时表单配置内容区滚动，底部按钮区固定不滚动 |

参考抽屉规范，表单配置内容区左右内间距为 20px，加上内容卡片自身的 12px 内间距，视觉总间距为 32px，与抽屉内容区 32px padding 保持一致。

#### 底部按钮区 (Footer)

| 属性 | 规范值 |
|------|--------|
| 高度 | 56px |
| 按钮对齐 | 左对齐 |
| 默认按钮 | "保存"（主按钮） |
| 按钮高度 | 32px |
| 按钮字号 | 14px |
| 按钮左侧间距 | 0（无额外 padding，按钮左边缘与内容卡片内容区左边缘对齐，视觉间距 12px 由卡片提供） |
| 顶部分割线 | 无 |
| 滚动 | 固定不滚动 |

#### 表单配置页验收标准

- 页面标题/页头高度为 48px
- 表单配置内容区左右内间距为 20px，加上卡片 12px，视觉总间距为 32px
- 表单配置内容区遵循 design-form.md 表单规范
- 底部按钮区高度为 56px
- 底部按钮左对齐，默认为"保存"主按钮
- 按钮高度 32px，字号 14px
- 按钮左侧无额外 padding
- 内容超出时仅表单配置内容区滚动，底部按钮区固定
- 底部按钮区无顶部分割线

#### 页面标题/页头

- 高度 48px，默认为下钻页样式
- 标题文案固定格式：**新增XX**（XX 为业务对象名称，如"新增策略"、"新增规则"）
- 其他规范与表单配置页一致

#### 横向步骤条

- 遵循 [comp-stepper.md](comp-stepper.md) 横向步骤条规范
- 步骤条与上方页面标题/页头间距：8px
- 步骤条与下方表单配置内容区间距：8px
- 步骤条左右内间距与表单配置内容区一致（20px + 卡片 12px = 视觉总间距 32px）

#### 表单配置内容区

- 与表单配置页（9.3）完全一致
- 左右内间距 20px，加上卡片 12px，视觉总间距 32px
- 遵循 [design-form.md](design-form.md) 表单规范
- 每一步骤对应独立的表单内容，切换步骤时表单内容替换

#### 底部按钮区

| 属性 | 规范值 |
|------|--------|
| 高度 | 56px |
| 按钮对齐 | 左对齐 |
| 按钮高度 | 32px |
| 按钮字号 | 14px |
| 按钮左侧间距 | 0（与表单配置页一致） |
| 顶部分割线 | 无 |
| 滚动 | 固定不滚动 |

按钮组合随步骤变化：

| 当前步骤 | 按钮组合（从左到右） |
|----------|---------------------|
| 第一步 | 取消 + 下一步（主按钮） |
| 中间步骤 | 取消 + 上一步 + 下一步（主按钮） |
| 最后一步 | 取消 + 上一步 + 完成（主按钮） |

按钮间距：按钮之间间距 8px。

#### 步骤配置页验收标准

- 页面标题/页头高度为 48px，标题文案为"新增XX"格式
- 横向步骤条遵循 comp-stepper.md 规范
- 步骤条与标题栏间距 8px，与表单内容区间距 8px
- 表单配置内容区左右内间距为 20px，加上卡片 12px，视觉总间距为 32px
- 底部按钮区高度为 56px
- 第一步按钮：取消 + 下一步（主按钮）
- 中间步骤按钮：取消 + 上一步 + 下一步（主按钮）
- 最后一步按钮：取消 + 上一步 + 完成（主按钮）
- 按钮高度 32px，字号 14px，间距 8px
- 内容超出时仅表单配置内容区滚动，步骤条和底部按钮区固定
