# 纯左树导航 - 表单配置页

## 1. 页面结构

```
┌──────────┬───────────────────────────────────────────────────┐
│          │ 页面标题栏 (48px，紧贴左侧导航)                   │
│          ├───────────────────────────────────────────────────┤
│          │ ↓ 间距 8px                                        │
│  左侧    │ ┌─────────────────────────────────────────────┐  │
│  深色    │ │ 表单配置内容区                               │  │
│  导航    │ │ (左右视觉总间距 32px)                        │  │
│          │ ├─────────────────────────────────────────────┤  │
│          │ │ 底部按钮区 (56px)                            │  │
│          │ └─────────────────────────────────────────────┘  │
└──────────┴───────────────────────────────────────────────────┘
```

## 2. 核心模块规范

- **表单内容区**：左右内间距 20px（加上卡片 12px = 32px）。遵循 [design-form.md](../components/design-form.md)。
- **底部按钮区**：高度 56px，左对齐。

## 3. 验收标准
- 表单区独立滚动。
- 底部按钮左边缘与内容卡片内容区对齐（视觉间距 12px）。
