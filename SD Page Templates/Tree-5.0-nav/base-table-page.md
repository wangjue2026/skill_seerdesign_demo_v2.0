# 纯左树导航 - 基础表格页

## 1. 页面结构

```
┌──────────┬───────────────────────────────────────────────────┐
│          │ 页面标题栏 (48px，紧贴左侧导航)                   │
│          ├───────────────────────────────────────────────────┤
│          │ ↓ 间距 8px                                        │
│  左侧    │ ┌─────────────────────────────────────────────┐  │
│  深色    │ │ Toolbar (上边距 8px)                         │  │
│  导航    │ │ ↓ 间距 8px                                  │  │
│          │ │ Table                                        │  │
│          │ │ Pagination (吸附底部, 40px)                │  │
│          │ └─────────────────────────────────────────────┘  │
└──────────┴───────────────────────────────────────────────────┘
```

## 2. 核心模块规范

- **Toolbar**：左右分栏，左侧操作按钮组（32px），右侧筛选组（搜索框 240px，刷新按钮 32x32px）。
- **Table**：无外边框，左右内间距 0（由卡片提供），操作列冻结在右侧。
- **Pagination**：高度 40px，顶部分割线 1px (#E1E5EB)，靠右对齐。

## 3. 验收标准
- 页面内容卡片与左侧导航间距 12px。
- 内容卡片左右内间距 12px。
- 分页区固定吸附在卡片底部。
