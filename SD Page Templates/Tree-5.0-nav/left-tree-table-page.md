# 纯左树导航 - 左树表格页

## 1. 页面结构

左侧树与右侧内容区位于同一个页面内容卡片内。

```
┌──────────┬───────────────────────────────────────────────────────┐
│          │ 页面标题栏 (48px，紧贴左侧导航)                       │
│          ├───────────────────────────────────────────────────────┤
│          │ ↓ 间距 8px                                            │
│  左侧    │ ┌──────────────────────────────────────────────────┐│
│  深色    │ │ 左侧树   │ 右侧内容区                              ││
│  导航    │ │ (240px)  │ Toolbar + Table + Pagination            ││
│          │ └──────────────────────────────────────────────────┘│
└──────────┴───────────────────────────────────────────────────────┘
```

## 2. 核心模块规范

- **左侧树区域**：默认宽度 240px，中间以 1px 竖向分割线分隔。拥有独立滚动。
- **右侧内容区**：由 Toolbar、Table、Pagination 组成。
- **间距**：右侧内容区继承卡片 12px 内间距。

## 3. 验收标准
- 页面内容卡片与左侧导航间距 12px。
- 左侧树与右侧区域之间无额外 padding。
- 点击树节点，右侧表格刷新。
