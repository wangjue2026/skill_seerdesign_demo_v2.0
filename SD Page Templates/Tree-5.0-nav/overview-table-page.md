# 纯左树导航 - 概览表格页

## 1. 页面结构

```
┌──────────┬───────────────────────────────────────────────────────┐
│          │ 页面标题栏 (48px，紧贴左侧导航)                       │
│          ├───────────────────────────────────────────────────────┤
│          │ ↓ 间距 8px                                            │
│  左侧    │ 筛选区（右对齐：隐藏概览 + 日期 + 刷新）             │
│  深色    │ ↓ 间距 8px                                            │
│  导航    │ ┌─────────────────────────────────────────────────┐  │
│          │ │ 概览卡片 (高度 166px)                        │  │
│          │ └─────────────────────────────────────────────────┘  │
│          │ ↓ 间距 8px                                            │
│          │ ┌─────────────────────────────────────────────────┐  │
│          │ │ 表格内容卡片 (Toolbar + Table + Pagination)      │  │
│          │ └─────────────────────────────────────────────────┘  │
└──────────┴───────────────────────────────────────────────────────┘
```

## 2. 核心模块规范

- **概览卡片**：默认高度 166px，白色背景，2px 圆角。
- **显示/隐藏切换**："隐藏概览"按钮位于右侧筛选区，点击后概览卡片折叠，表格区自适应拉伸。
- **间距**：各区域垂直间距统一为 8px。

## 3. 验收标准
- 概览卡片与表格卡片之间间距为 8px。
- 隐藏概览后布局不得错位。
