# 纯左树导航框架通用规范

## 一、整体结构

右侧业务内容区由两个上下排列的模块组成：

1. 页面标题栏 (Page Title)
2. 页面内容卡片

### 布局层级

```
┌──────────┬────────────────────────────────────────────────────┐
│          │ A. 页面标题栏 (Page Title) - 48px, 白色, 无边框    │
│          ├────────────────────────────────────────────────────┤
│          │ ↓ 间距 8px                                       │
│  左侧    │ ┌──────────────────────────────────────────────┐ │
│  导航    │ │ B. 页面内容卡片 (白色, 圆角 2px, 无边框)      │ │
│          │ │                                              │ │
│          │ │  卡片内部的具体内容由页面类型定义               │ │
│          │ └──────────────────────────────────────────────┘ │
└──────────┴────────────────────────────────────────────────────┘
```

## 二、页面标题栏规范

- **高度**：48px。
- **与左侧导航关系**：紧贴左侧导航，间距 0px。
- **与浏览器右侧关系**：横向铺满，无外间距。
- **组件要求**：详见 [comp-page-title.md](../components/comp-page-title.md)。

## 三、页面内容卡片规范

### 3.1 基础样式
白色背景，圆角 2px，无边框。

### 3.2 外间距
- 与页面标题栏：8px。
- 与左侧导航：12px。
- 与浏览器右侧：12px。
- 与浏览器底部：8px。

### 3.3 内间距
- 左右：12px。
- 上下：0px。

### 3.4 业务内容对齐
所有一级业务模块（Toolbar/Table/Pagination）左右视觉边缘应与卡片内容区边界对齐，间距只由页面内容卡片提供。

## 四、设计原则与禁止项

### 设计原则
- **标题联动原则**：标题栏文字与左侧菜单名称一致。
- **间距独立原则**：卡片使用 margin 与标题栏和导航分隔。

### 禁止项
- 禁止标题栏与左侧导航之间出现空白。
- 禁止内容卡片紧贴左侧导航（应有 12px 间距）。
- 禁止内容卡片内部业务内容通栏贴到边缘。

## 五、验收标准
1. 页面标题栏高度 48px，与导航 0 间距。
2. 页面内容卡片上外间距 8px，左右外间距 12px，下外间距 8px。
3. 页面内容卡片左右内间距 12px。
