# L 型导航框架通用规范

## 一、整体结构

L 型导航页面的内容区由一个外层容器包裹，内部为内容卡片。

内容卡片内部包含：

1. 页面标题/页头
2. 业务内容区（操作与筛选区、表格、分页等）

### 布局层级

```
┌─────────────────────────────────────────────────────────────────┐
│ 外层容器 (背景色: #F7F8FA, padding: 12px 或无)                   │
│ ┌─────────────────────────────────────────────────────────────┐ │
│ │ 内容卡片 (白色, 圆角 2px)                                   │ │
│ │ ┌─────────────────────────────────────────────────────────┐ │ │
│ │ │ A. 页面标题/页头 (Page Title) - 48px                     │ │ │
│ │ └─────────────────────────────────────────────────────────┘ │ │
│ │                     ↓ 间距 8px                              │ │
│ │ ┌─────────────────────────────────────────────────────────┐ │ │
│ │ │ B. 操作与筛选区 (Toolbar)                                │ │ │
│ │ └─────────────────────────────────────────────────────────┘ │ │
│ │                     ↓ 间距 8px                              │ │
│ │ ┌─────────────────────────────────────────────────────────┐ │ │
│ │ │ C. 数据展示区 (Table)                                    │ │ │
│ │ └─────────────────────────────────────────────────────────┘ │ │
│ │                     ↓                                       │ │
│ │ ┌─────────────────────────────────────────────────────────┐ │ │
│ │ │ D. 分页区 (Pagination) - 40px                           │ │ │
│ │ └─────────────────────────────────────────────────────────┘ │ │
│ └─────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
```

### 垂直堆叠顺序

| 区域 | 说明 |
|-----|------|
| A. 页面标题/页头 (Page Title) | 位于内容卡片最上方，全宽 |
| B. 操作与筛选区 (Toolbar) | 页面标题/页头下方，与页面标题/页头间距 8px |
| C. 数据展示区 (Table) | 操作与筛选区下方，与工具栏间距 8px |
| D. 分页区 (Pagination) | 位于容器底部 |

## 二、外层容器规范

| 属性 | 规范值 |
|------|--------|
| 背景色 | var(--color-graphite-l40) / #F7F8FA |
| padding | 根据页面类型确定（基础表格页 12px 四周；左树表格页无 padding） |

## 三、页面标题/页头规范

### 3.1 基础样式

页面标题/页头高度固定为 48px。
页面标题/页头必须位于内容卡片顶部。

### 3.2 左边距

- 页面标题/页头的左边距为 16px（相对于内容卡片）
- 内容卡片拥有左右 12px 内间距，因此标题文字实际左边距为 28px（12px + 16px）

### 3.3 组件使用要求

页面标题/页头规范详见 [comp-page-title.md](../components/comp-page-title.md)

## 四、页面内容卡片规范

### 4.1 基础样式

| 属性 | 规范值 |
|------|--------|
| 背景色 | 纯白 (#FFFFFF) |
| 圆角 | 2px |
| 边框 | 无 |

### 4.2 内间距

| 属性 | 规范值 |
|------|--------|
| 左右内间距 | 12px（padding: 0 12px） |
| 上下内间距 | 无 |

### 4.3 内容放置规则

所有业务内容必须放置在内容卡片内。

业务内容包括但不限于：筛选区、操作按钮区、表格、表单、图表、详情信息、分页。

内容卡片内部不允许再次出现页面标题/页头，也不允许重复展示当前页面标题。

### 4.4 业务内容与卡片边缘的关系

内容卡片内部的业务内容不得通栏贴到卡片左右边缘。
业务内容的左侧视觉边缘必须距离内容卡片左边缘 12px。
业务内容的右侧视觉边缘必须距离内容卡片右边缘 12px。

### 4.5 卡片内边距唯一来源原则

内容卡片统一定义左右内间距为 12px。子模块不再单独设置左右 padding 或 margin。

## 五、设计原则与禁止项

### 设计原则

- **全屏原则**：内容卡片高度占满屏幕。
- **统一间距原则**：所有区域间距统一为 8px。
- **分页固定原则**：分页区吸附在内容卡片底部。

### 禁止项

- 禁止页面内容卡片内部业务内容通栏贴到卡片左右边缘。
- 禁止子模块（Toolbar/Table/Pagination）重复设置左右 padding。
- 禁止改变页面标题/页头高度（固定 48px）。

## 六、验收标准

1. 外层容器背景色为 #F7F8FA。
2. 内容卡片背景为白色，圆角为 2px，无边框。
3. 内容卡片左右内间距为 12px，上下内间距为 0px。
4. 页面标题/页头高度固定为 48px。
5. Toolbar 与页面标题/页头间距为 8px。
6. 分页区高度固定为 40px，顶部有分割线。
