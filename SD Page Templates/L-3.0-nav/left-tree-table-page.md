# L 型导航 - 左树表格页

## 1. 布局层级

```
┌─────────────────────────────────────────────────────────────────┐
│ 外层容器 (背景色: #F7F8FA, 无 padding)                          │
│ ┌─────────────────────────────────────────────────────────────┐ │
│ │ 内容卡片 (白色, 圆角 2px)                                   │ │
│ │ ┌─────────────────────────────────────────────────────────┐ │ │
│ │ │ A. 页面标题/页头 (Page Title) - 48px                     │ │ │
│ │ └─────────────────────────────────────────────────────────┘ │ │
│ │ ↓ 无间距                                                     │ │
│ │ ┌──────────────────┐ ┌────────────────────────────────────┐│ │
│ │ │ B. 左侧内容区    │12│ C. 操作与筛选区 (Toolbar)         ││ │
│ │ │    240px         │p│                                     ││ │
```

## 2. 垂直堆叠与间距

- **外层容器**：无 padding（不设置 padding）。
- **页面标题/页头**：位于最上方，全宽。
- **左侧内容区 (Tree/Menu)**：宽度固定 240px，无间距紧跟在标题下方。
- **左侧与右侧间距**：12px。
- **右侧内部间距**：8px。

## 3. 左侧内容区 (Left Content Area)

- 宽度固定为 240px，不可伸缩。
- 右边框：1px 分割线 (#E1E5EB)。
- 树组件规范：节点高度 32px，选中态蓝色渐变背景 + 3px 左侧激活条。

## 4. 右侧内容区 (Right Content Area)

- 包含操作与筛选区、数据展示区、分页区。
- 宽度：自动填充剩余空间（最小宽度 600px）。

## 5. 验收标准
- 页面标题/页头横跨左右两个区域。
- 点击左侧树节点，右侧内容跟随变化。
- 左侧区域右侧有 1px 竖向分割线。
