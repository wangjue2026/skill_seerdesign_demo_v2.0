# L 型导航 - 表单配置页

## 1. 布局层级

```
┌─────────────────────────────────────────────────────────────────┐
│ 外层容器 (背景色: #F7F8FA, padding: 12px)                       │
│ ┌─────────────────────────────────────────────────────────────┐ │
│ │ 内容卡片 (白色, 圆角 2px)                                   │ │
│ │ ┌─────────────────────────────────────────────────────────┐ │ │
│ │ │ A. 页面标题/页头 (Page Title) - 48px                     │ │ │
│ │ └─────────────────────────────────────────────────────────┘ │ │
│ │                     ↓ 间距 8px                              │ │
│ │ ┌─────────────────────────────────────────────────────────┐ │ │
│ │ │ B. 表单配置内容区 (Form Content)                         │ │ │
│ │ │    左右内间距 20px，遵循表单规范                           │ │ │
│ │ └─────────────────────────────────────────────────────────┘ │ │
│ │                     ↓                                       │ │
│ │ ┌─────────────────────────────────────────────────────────┐ │ │
│ │ │ C. 底部按钮区 (Footer) - 56px                           │ │ │
│ │ │    按钮左对齐，默认"保存"按钮（主按钮）                    │ │ │
│ │ └─────────────────────────────────────────────────────────┘ │ │
│ └─────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
```

## 2. 表单配置内容区 (Form Content)

- **左右内间距**：20px（加上卡片 12px，视觉总间距 32px）。
- **表单规范**：遵循 [design-form.md](../components/design-form.md)。
- **滚动**：内容超出时表单区滚动，底部按钮区固定。

## 3. 底部按钮区 (Footer)

- **高度**：56px。
- **对齐**：左对齐。
- **默认按钮**："保存"（主按钮，高度 32px）。
- **顶部分割线**：无。

## 4. 验收标准
- 视觉总间距（左右）为 32px。
- 底部按钮左对齐，视觉间距 12px 由卡片提供。
- 表单内容区独立滚动。
