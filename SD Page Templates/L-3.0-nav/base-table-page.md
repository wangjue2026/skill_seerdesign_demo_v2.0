# L 型导航 - 基础表格页

## 1. 布局层级

```
┌─────────────────────────────────────────────────────────────────┐
│ 外层容器 (背景色: #F7F8FA, padding: 12px)                       │
│ ┌─────────────────────────────────────────────────────────────┐ │
│ │ 内容卡片 (白色, 圆角 2px)                                   │ │
│ │ ┌─────────────────────────────────────────────────────────┐ │ │
│ │ │ A. 页面标题/页头 (Page Title) - 48px                     │ │ │
│ │ └─────────────────────────────────────────────────────────┘ │ │
│ │                     ↓ 间距 8px                              │ │
│ │ ┌─────────────────────────────────────────────────────────┐ │ │
│ │ │ B. 操作与筛选区 (Toolbar)                                │ │ │
│ │ └─────────────────────────────────────────────────────────┘ │ │
│ │                     ↓ 间距 8px                              │ │
│ │ ┌─────────────────────────────────────────────────────────┐ │ │
│ │ │ C. 数据展示区 (Table)                                    │ │ │
│ │ └─────────────────────────────────────────────────────────┘ │ │
│ │                     ↓                                       │ │
│ │ ┌─────────────────────────────────────────────────────────┐ │ │
│ │ │ D. 分页区 (Pagination) - 40px                           │ │ │
│ │ └─────────────────────────────────────────────────────────┘ │ │
│ └─────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
```

## 2. 操作与筛选区 (Toolbar)

**间距规范**：
- 与页面标题/页头间距：8px
- 与数据展示区间距：8px
- 左右内间距：0（继承内容卡片的 12px 内间距）

**结构规则**：
- 左侧：操作按钮组（第一个为主按钮，高度 32px，间距 8px）
- 右侧：筛选组（搜索框 240px，刷新按钮 32x32px 带边框，间距 8px）

## 3. 表格区 (Table)

- 与 Toolbar 间距：8px
- 外边框线：无
- 左右内间距：0（继承内容卡片的 12px 内间距）
- 横向滚动：内容超出时出现横向滚动条，操作列冻结在右侧。

## 4. 分页区 (Pagination)

- 位置：吸附在内容卡片底部
- 高度：40px
- 顶部分割线：1px, #E1E5EB
- 对齐方式：靠右对齐

## 5. 验收标准
- 外部 padding 为 12px。
- 刷新按钮必须为 32x32px 外框，带边框线。
- 表格无外边框。
