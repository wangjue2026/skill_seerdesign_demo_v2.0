# L 型导航 - 步骤配置页

## 1. 布局层级

```
┌─────────────────────────────────────────────────────────────────┐
│ 外层容器 (背景色: #F7F8FA, padding: 12px)                       │
│ ┌─────────────────────────────────────────────────────────────┐ │
│ │ 内容卡片 (白色, 圆角 2px)                                   │ │
│ │ ┌─────────────────────────────────────────────────────────┐ │ │
│ │ │ A. 页面标题/页头 (Page Title) - 48px                     │ │ │
│ │ └─────────────────────────────────────────────────────────┘ │ │
│ │                     ↓ 间距 8px                              │ │
│ │ ┌─────────────────────────────────────────────────────────┐ │ │
│ │ │ B. 横向步骤条 (Stepper)                                   │ │ │
│ │ └─────────────────────────────────────────────────────────┘ │ │
│ │                     ↓ 间距 8px                              │ │
│ │ ┌─────────────────────────────────────────────────────────┐ │ │
│ │ │ C. 表单配置内容区 (Form Content)                         │ │ │
│ │ └─────────────────────────────────────────────────────────┘ │ │
│ │                     ↓                                       │ │
│ │ ┌─────────────────────────────────────────────────────────┐ │ │
│ │ │ D. 底部按钮区 (Footer) - 56px                           │ │ │
│ │ └─────────────────────────────────────────────────────────┘ │ │
│ └─────────────────────────────────────────────────────────────┘ │
└─────────────────────────────────────────────────────────────────┘
```

## 2. 核心组件规范

- **页面标题/页头**：固定格式为"新增XX"。
- **横向步骤条**：遵循 [comp-stepper.md](../components/comp-stepper.md)。
- **表单配置内容区**：左右视觉间距 32px，遵循表单规范。

## 3. 底部按钮逻辑

| 当前步骤 | 按钮组合（从左到右） |
|----------|---------------------|
| 第一步 | 取消 + 下一步（主按钮） |
| 中间步骤 | 取消 + 上一步 + 下一步（主按钮） |
| 最后一步 | 取消 + 上一步 + 完成（主按钮） |

- 按钮高度 32px，间距 8px。

## 4. 验收标准
- 步骤条与标题栏、表单内容区间距均为 8px。
- 表单内容区随步骤切换内容。
- 仅表单内容区滚动，标题、步骤条、底部按钮固定。
