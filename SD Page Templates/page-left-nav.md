# 左侧导航页面通用内容模块规范

## 适用范围

适用于左侧深色导航 + 右侧业务内容区的页面模板。

本规范只定义右侧业务内容区中的两个通用模块：

1. 页面标题栏
2. 页面内容卡片

不定义左侧导航自身样式，不定义表格、表单、筛选区、图表等业务组件样式。

---

## 一、整体结构

右侧业务内容区必须由两个上下排列的模块组成：

1. 页面标题栏
2. 页面内容卡片

页面标题栏与页面内容卡片是同级模块，不允许将页面内容卡片包裹在页面标题栏内，也不允许将页面标题栏放入页面内容卡片内。

页面内容卡片内部只能放置业务内容，例如筛选区、按钮区、表格、表单、图表、详情信息、分页等。

### 布局层级

```
┌──────────┬────────────────────────────────────────────────────┐
│          │ A. 页面标题栏 (Page Title) - 48px, 白色, 无边框    │
│          ├────────────────────────────────────────────────────┤
│          │ ↓ 间距 8px                                       │
│  左侧    │ ┌──────────────────────────────────────────────┐ │
│  导航    │ │ B. 页面内容卡片 (白色, 圆角 2px, 无边框)      │ │
│          │ │                                              │ │
│          │ │  卡片内部的具体内容由页面类型定义               │ │
│          │ │  详见第八部分"页面类型定义"                    │ │
│          │ └──────────────────────────────────────────────┘ │
└──────────┴────────────────────────────────────────────────────┘
```

页面内容卡片内部的具体布局（Toolbar、Table、Pagination、表单、步骤条等）由第八部分"页面类型定义"中各页面类型单独定义，本章节仅定义到页面内容卡片层级。

### 整体容器规范

| 属性 | 规范值 |
|------|--------|
| 背景色 | var(--color-graphite-l50) / #F7F9FC |
| 布局 | flex 横向排列（左侧导航 + 右侧内容区） |
| 高度 | 100vh |

### 左侧导航与右侧内容区间距

| 属性 | 规范值 |
|------|--------|
| 左侧导航与右侧内容区 | 无间距（导航右侧直接衔接右侧内容区） |

## 二、页面标题栏规范

### 2.1 基础样式

页面标题栏用于展示当前页面标题。

页面标题栏高度固定为 48px，背景色为白色，无边框，无底部分割线。

页面标题栏必须位于右侧业务内容区顶部。

### 2.2 与左侧导航的关系

页面标题栏必须紧贴左侧导航。

页面标题栏与左侧导航之间的间距为 0px。

不得在页面标题栏外侧增加左边距、左内边距、空白占位或背景间隔，导致标题栏整体与左侧导航之间出现空隙。

### 2.3 与浏览器右侧的关系

页面标题栏应横向铺满右侧业务内容区。

页面标题栏右侧与浏览器右边缘之间不需要额外外间距。

### 2.4 标题内容

页面标题栏中的标题文字必须与当前左侧导航选中的菜单名称保持一致。

如果当前页面由左侧导航的二级菜单进入，则标题文字应显示当前选中的二级菜单名称。

页面标题栏不得为空，不得显示路由路径、英文 key、组件名或占位文案。

### 2.5 组件使用要求

页面标题栏必须使用基础标题栏组件规范（详见 [comp-page-title.md](comp-page-title.md)）。

如果项目中已存在统一的页面标题栏组件，应优先使用该组件。

不允许为了快速实现而自行使用普通文字、普通容器或临时样式模拟标题栏。

### 2.6 左侧导航页面中的使用说明

- 页面标题栏的左边距为 12px（相对于内容卡片）
- 返回图标与标题文字间距为 12px
- 返回图标使用向左箭头图标
- 图标左间距为 12px
- 与左侧导航无间距，吸附在顶部

## 三、页面内容卡片规范

### 3.1 基础样式

页面内容卡片是承载业务内容的白色卡片区域。

页面内容卡片背景色为白色，圆角为 2px，无边框。

页面内容卡片位于页面标题栏下方。

### 3.2 外间距

页面内容卡片与页面标题栏之间的间距为 8px。

页面内容卡片与左侧导航之间的间距为 12px。

页面内容卡片与浏览器右侧边缘之间的间距为 12px。

页面内容卡片与浏览器底部之间的间距为 8px。

需要注意：页面标题栏与左侧导航之间没有间距，但页面内容卡片与左侧导航之间必须有 12px 间距。

### 3.3 内间距

页面内容卡片自身的内间距为：

- 上内间距：0px
- 下内间距：0px
- 左内间距：12px
- 右内间距：12px

不得使用组件库默认卡片内边距覆盖该规范。

禁止出现上下 16px、左右 20px 等默认卡片内间距。

### 3.4 内容放置规则

所有业务内容必须放置在页面内容卡片的内容区域内。

业务内容包括但不限于：

- 筛选区
- 操作按钮区
- 表格
- 表单
- 图表
- 详情信息
- 分页

页面内容卡片内部不允许再次出现页面标题栏，也不允许重复展示当前页面标题。

### 3.5 业务内容与卡片边缘的关系

页面内容卡片内部的业务内容不得通栏贴到卡片左右边缘。

业务内容的左侧视觉边缘必须距离页面内容卡片左边缘 12px。

业务内容的右侧视觉边缘必须距离页面内容卡片右边缘 12px。

例如表格放入页面内容卡片时，表格不得直接贴齐卡片左右边缘，必须受到卡片左右 12px 内间距约束。

### 3.6 卡片内边距唯一来源原则

页面内容卡片已经统一定义左右内间距为 12px、上下内间距为 0px。

因此，页面内容卡片内部的一级业务模块不再单独设置左右 padding、左右 margin 或额外容器间距。

适用模块包括但不限于：

- Toolbar 操作筛选区
- Table 表格区
- Pagination 分页区
- Form 表单区
- Chart 图表区

这些模块在水平方向上应充满页面内容卡片的可用内容区，而不是充满卡片外边界。

视觉结果应表现为：

- 按钮左边缘距离内容卡片左边缘 12px；
- 表格左边缘距离内容卡片左边缘 12px；
- 分页器右边缘距离内容卡片右边缘 12px；
- 这些 12px 间距只能来自页面内容卡片自身的左右内间距；
- 不允许在 Toolbar、Table、Pagination 等子模块上重复设置左右 padding 或 margin。

禁止出现以下情况：

1. 页面内容卡片已有左右 12px 内间距，Toolbar 又额外设置左右 12px padding。
2. 页面内容卡片已有左右 12px 内间距，Table 又额外设置左右 12px padding。
3. 页面内容卡片已有左右 12px 内间距，Pagination 又额外设置左右 12px padding。
4. 子模块因重复设置间距，导致视觉间距变成 24px 或更大。
5. 为了让表格贴边而取消页面内容卡片的统一左右内间距。

验收标准：

页面内容卡片内部所有一级业务模块的视觉左右边缘，应与卡片内容区边界对齐。
最终视觉效果为：业务内容距离页面内容卡片左右边缘均为 12px，且该间距只由页面内容卡片提供。

## 四、布局关系总结

左侧导航与右侧业务内容区整体可以直接相邻。

页面标题栏作为右侧业务内容区顶部模块，必须与左侧导航无间距衔接。

页面内容卡片作为标题栏下方模块，必须相对于左侧导航保留 12px 左外间距。

页面内容卡片相对于页面标题栏保留 8px 上外间距。

页面内容卡片相对于浏览器右侧保留 12px 右外间距。

页面内容卡片相对于浏览器底部保留 8px 下外间距。

页面内容卡片内部相对于业务内容保留左右 12px 内间距，上下不设置内间距。

---

## 五、设计原则

| 原则 | 说明 |
|------|------|
| 标题联动原则 | 页面标题栏文字与左侧导航选中菜单名称保持一致 |
| 间距独立原则 | 内容卡片使用 margin 与页面标题栏和左侧导航分隔，不依赖 padding |
| 卡片独立原则 | 内容卡片自身无上下内间距，子元素各自管理间距 |
| 分页吸附原则 | 分页区吸附在内容卡片底部 |
| 导航适配原则 | 适用于 DR、SASE、XDR 等纯左侧导航产品 |

## 六、禁止项

禁止出现以下实现：

1. 页面标题栏与左侧导航之间出现空白间距。
2. 页面标题栏未使用统一基础标题栏组件，而是临时手写标题文字。
3. 页面标题栏为空，或标题显示为路由路径、英文 key、占位文字。
4. 页面内容卡片紧贴左侧导航。
5. 页面内容卡片紧贴浏览器右侧边缘。
6. 页面内容卡片紧贴浏览器底部。
7. 页面内容卡片内部出现重复页面标题。
8. 页面内容卡片内部业务内容通栏贴到卡片左右边缘。
9. 页面内容卡片使用组件库默认内间距，例如上下 16px、左右 20px。
10. 通过删除页面标题栏、隐藏页面标题栏、合并标题栏和内容卡片的方式解决布局问题。

---

## 七、验收标准

页面实现完成后，必须满足以下标准：

1. 页面标题栏高度为 48px。
2. 页面标题栏背景为白色。
3. 页面标题栏无边框、无底部分割线。
4. 页面标题栏与左侧导航之间的间距为 0px。
5. 页面标题栏标题文字与当前左侧导航选中菜单名称一致。
6. 页面标题栏使用基础标题栏组件规范。
7. 功能全览和快捷入口页面不需要页面标题栏。
8. 页面内容卡片背景为白色，圆角 2px，无边框。
9. 页面内容卡片上外间距 8px（与页面标题栏间距）。
10. 页面内容卡片左右外间距 12px（与左侧导航间距）。
11. 页面内容卡片下外间距 8px。
12. 页面内容卡片左右内间距 12px，上下内间距 0px。
13. 页面内容卡片内部业务内容左右受 12px 内间距约束。
14. 禁止在内容卡片上添加边框线。
15. 禁止内容卡片设置上下内间距。
16. 禁止页面标题栏添加底部分割线。

## 八、页面类型定义

页面内容卡片内部由操作与筛选区（Toolbar）、数据展示区（Table）、分页区（Pagination）等模块组成。

根据不同业务场景，页面内容卡片内可组合为不同的页面类型。

### 8.1 基础表格页

适用场景：
- 标准的列表管理页面
- 设备列表、策略列表、用户列表等

#### 页面结构

```
┌──────────┬───────────────────────────────────────────────────┐
│          │ 页面标题栏 (48px，紧贴左侧导航)                   │
│          ├───────────────────────────────────────────────────┤
│          │ ↓ 间距 8px                                        │
│  左侧    │ ┌─────────────────────────────────────────────┐  │
│  深色    │ │ Toolbar (上边距 8px, 无左右内间距)           │  │
│  导航    │ │ ↓ 间距 8px                                  │  │
│          │ │ Table (无左右内间距)                         │  │
│          │ │ Pagination (吸附底部, 40px, 顶部分割线 1px) │  │
│          │ └─────────────────────────────────────────────┘  │
│          │         ← 12px →                    ← 12px →     │
└──────────┴───────────────────────────────────────────────────┘
```

说明：
- 左侧深色导航与页面标题栏无间距
- 页面内容卡片与左侧导航间距 12px
- 页面内容卡片与浏览器右侧间距 12px
- 页面内容卡片左右内间距 12px（Toolbar/Table/Pagination 无额外左右间距）
- 页面内容卡片内部从上到下依次为：Toolbar → 8px 间距 → Table → Pagination

#### 操作与筛选区 (Toolbar)

间距规范：

| 属性 | 规范值 |
|------|--------|
| 上边距 | 8px |
| 与表格区间距 | 8px |
| 左右内间距 | 无（toolbar 左右边缘与卡片内容区左右边缘对齐） |

结构规则：

Toolbar 左右分栏：
- 左侧为操作按钮组
- 右侧为筛选与快捷操作组

左侧按钮组 (Action Group)：
- 第一个按钮为主按钮（Primary），用于页面主操作，例如"新建"
- 后续按钮为普通按钮（Default），例如"导出""批量操作"
- 按钮高度统一为 32px
- 按钮之间水平间距为 8px
- 按钮图标为可选项，由当前组件库映射

右侧筛选组 (Filter Group)：
- 排列顺序（从左到右固定）：日期筛选器 → 选择器 → 搜索框 → 刷新按钮
- 搜索框宽度为 240px
- 刷新按钮尺寸为 32px × 32px
- 刷新按钮需要有边框（边框颜色 graphite-l30, #E1E5EB）
- 筛选项之间水平间距为 8px

#### 数据展示区 (Table)

结构规则：

| 属性 | 规范值 |
|------|--------|
| 与操作与筛选区间距 | 8px |
| 外边框线 | 无 |
| 左右内间距 | 无（table 左右边缘与卡片内容区左右边缘对齐） |
| 横向滚动 | 内容展示不完整时出现横向滚动条，不要拉高表格高度 |
| 操作列冻结 | 有横向滚动条时，操作列冻结在右侧 |

#### 分页区 (Pagination)

结构规则：

| 属性 | 规范值 |
|------|--------|
| 位置 | 吸附在页面内容卡片底部 |
| 高度 | 40px |
| 顶部分割线 | 1px, graphite-l30 (#E1E5EB) |
| 左右内间距 | 无（分页区左右边缘与卡片内容区左右边缘对齐） |
| 对齐方式 | 靠右对齐 |

#### 验收标准

操作与筛选区：
- 左侧按钮组靠左对齐
- 右侧筛选组靠右对齐
- 左右区域垂直居中
- Toolbar 上边距 8px
- Toolbar 与表格区域间距 8px

数据展示区：
- 表格区域无外边框线
- 表格高度固定，内容超出时横向滚动
- 有横向滚动时操作列固定可见

分页区：
- 分页区高度固定为 40px
- 顶部有 1px 分割线
- 分页控件靠右对齐

### 8.2 左树表格页

适用场景：
- 需要按组织/分组/区域筛选的列表页面
- 设备列表（按分组）、策略列表（按类型）等

#### 页面结构

```
┌──────────┬───────────────────────────────────────────────────────┐
│          │ 页面标题栏 (48px，紧贴左侧导航)                       │
│          ├───────────────────────────────────────────────────────┤
│          │ ↓ 间距 8px                                            │
│  左侧    │ ┌──────────────────────────────────────────────────┐│
│  深色    │ │ 左侧树   │ 右侧内容区                              ││
│  导航    │ │ (240px)  │ Toolbar (上边距 8px)                    ││
│          │ │          │ ↓ 间距 8px                               ││
│          │ │          │ Table                                    ││
│          │ │          │ Pagination (吸附底部, 40px, 1px 分割线) ││
│          │ └──────────────────────────────────────────────────┘│
│          │         ← 12px →                       ← 12px →      │
└──────────┴───────────────────────────────────────────────────────┘
```

说明：
- 左侧深色导航与页面标题栏无间距
- 页面内容卡片与左侧导航间距 12px
- 页面内容卡片与浏览器右侧间距 12px
- 左侧树与右侧内容区在同一个页面内容卡片内，中间无间距，仅靠 1px 竖向分割线（graphite-l30）分隔

#### 左侧树区域

- 左侧树宽度默认 240px
- 左侧树与右侧内容区之间无间距，仅靠 1px 竖向分割线（graphite-l30）分隔
- 树区域有独立的竖向滚动
- 树区域上下无额外间距

#### 右侧内容区

右侧内容区由操作与筛选区（Toolbar）、数据展示区（Table）、分页区（Pagination）组成。

Toolbar：
- 上边距 8px
- 左右无额外内间距（由卡片 12px 内间距提供）
- 按钮组、筛选组规则与基础表格页一致

Table：
- 与 Toolbar 间距 8px
- 左右无额外内间距
- 横向滚动时操作列冻结在右侧

Pagination：
- 吸附在页面内容卡片底部
- 高度 40px，顶部分割线 1px graphite-l30
- 右对齐

### 8.3 表单配置页

适用场景：
- 独立的配置表单页面
- 系统设置、策略配置、规则配置等

#### 页面结构

```
┌──────────┬───────────────────────────────────────────────────┐
│          │ 页面标题栏 (48px，紧贴左侧导航)                   │
│          ├───────────────────────────────────────────────────┤
│          │ ↓ 间距 8px                                        │
│  左侧    │ ┌─────────────────────────────────────────────┐  │
│  深色    │ │ B: 表单配置内容区                            │  │
│  导航    │ │ (左右 20px + 卡片 12px = 视觉总间距 32px)   │  │
│          │ │                                             │  │
│          │ │ 遵循 design-form.md 表单规范                 │  │
│          │ │                                             │  │
│          │ ├─────────────────────────────────────────────┤  │
│          │ │ C: 底部按钮区 (56px，按钮左对齐)            │  │
```

### 8.4 概览表格页

适用场景：
- 需要在表格上方展示统计概览数据的页面
- 安全总览、监控总览、资产概览等

#### 页面结构

```
│  深色    │                                                       │
```

说明：
- 左侧深色导航与页面标题栏无间距

- 筛选区、概览卡片、表格卡片之间间距均为 8px

#### 布局层级说明

| 层级 | 模块 | 说明 |
|------|------|------|

| C | 概览卡片 | 与筛选区间距 8px；默认高度 166px（可根据内容增高）；白色背景，2px 圆角，遵循内容卡片通用规范；支持"隐藏概览/显示概览"切换 |
| D | 表格内容卡片 | 与概览卡片间距 8px；完整基础表格页结构（Toolbar + Table + Pagination） |

#### 筛选区 (Filter Bar)

| 属性 | 规范值 |

| 对齐方式 | 右对齐 |
| 典型内容 | "隐藏概览"按钮 + 日期范围筛选器 + 刷新 icon 按钮 |

#### 概览显示/隐藏切换

筛选区默认包含"隐藏概览"按钮，用于切换概览卡片的显示与隐藏。

| 属性 | 规范值 |
|------|--------|
| 按钮类型 | 普通按钮 |
| 按钮样式 | 遵循 comp-button.md 普通按钮规范 |
| 按钮位置 | 日期范围筛选器左侧 |
| 概览显示时文案 | "隐藏概览" |
| 概览隐藏时文案 | "显示概览" |
| 默认状态 | 概览卡片显示 |

切换行为：
- 点击"隐藏概览"：概览卡片隐藏，表格内容卡片自适应增高，占据原概览卡片的高度和间距空间
- 点击"显示概览"：概览卡片恢复显示，表格内容卡片恢复原有高度
- 切换过程不得导致页面布局错位或抖动

#### 概览卡片 (Overview Card)

| 滚动 | 概览卡片高度默认 166px，内容超出时允许增高，不独立滚动 |

#### 表格内容卡片 (Table Card)

表格内容卡片遵循 8.1 基础表格页的全部规范，包含 Toolbar + Table + Pagination。

- 表格内容卡片遵循基础表格页规范
- "隐藏概览"按钮位于日期范围筛选器左侧
- 隐藏概览后表格内容卡片自适应增高，显示概览后恢复原有高度
