# 抽屉组件设计规范

## ⚠️ 高频违规速查（阅读正文前必看）

> AI 在生成抽屉代码前，必须先逐行核对此表，再阅读下方完整规范。

| 规则 | ✅ 正确 | ❌ 常见错误 |
|------|--------|------------|
| **默认宽度** | `800px` — 未指定时一律使用默认值 | 随意选用 640px 或其他非默认尺寸 |
| 宽度可选范围 | 仅可选 `720 / 800 / 960 / 1200px` | 使用规范之外的任意像素值 |
| 标题栏高度 | 固定 `48px`，不可撑高 | 高度未锁定，随内容变化 |
| 底部按钮栏高度 | 固定 `56px`，不可撑高 | 高度未锁定或使用 `48px` |
| 底部按钮对齐方向 | **左对齐**，主按钮 `margin-left: 16px`（容器内边距 `16px`，合计距边缘 `32px` 以便与内容对齐） | 右对齐或居中，或者未加容器内边距导致偏移量不对 |
| 按钮间距 | `8px` | 使用 `12px` 或 `16px` |
| 内容区左右边距 | `32px` | 使用 `16px` 或 `24px` |
| 标题栏底部分割线 | **无分割线** | 加了 `border-bottom` |
| 底部栏顶部分割线 | **有 1px 分割线**（使用 `#E1E5EB` / `graphite-l30`，无底色） | 无分割线，或使用了独立的底色（如 `#F7F7F8`） |
| 关闭按钮位置 | 标题栏**最右侧** | 放在标题文字左侧 |
| 底部按钮字号 | `14px` | 沿用默认 `12px` |
| 内容区溢出处理 | 仅内容区滚动，标题栏和底部栏**固定不动** | 整个抽屉一起滚动 |

## 使用声明

本规范只定义抽屉的视觉、布局、交互和验收规则，不提供具体框架代码实现。

生成代码时，必须以当前项目的技术栈 Skill 为准。

如果本规范与项目技术栈约束发生冲突，优先遵守项目技术栈约束。

禁止在本规范中包含：
- React / Vue / Angular 代码示例
- JSX / TSX
- TypeScript 示例
- import 语句
- 具体组件库 API
- 具体组件库内部 class 的全局覆盖
- open / styles / footer 等特定版本 API 示例

本规范中的 Drawer、Footer、Button 等名称仅表示设计语义，不代表必须使用某个框架或组件库的同名 API。

## 核心规范

- 默认从右侧滑出
- 标题栏高度：48px
- 底部按钮栏高度：56px
- 默认宽度：800px
- 按钮区域：左对齐
- 按钮之间间距：8px
- 内容区域左右边距：32px
- 内容超出时，在内容区域内部滚动
- 抽屉宽度只能从以下尺寸中选择：
  - 720px
  - 800px
  - 960px
  - 1200px

## 结构

抽屉从上到下分为三层：

1. 标题栏
2. 内容区域
3. 底部按钮栏

## 标题栏

- 高度固定为 48px
- 标题文字垂直居中且左对齐
- 标题文字左边距 16px
- 标题文字字号 16px，字重 600
- 无底部分割线
- **关闭按钮位置**：关闭按钮（"X" 或 Close Icon）必须位于标题栏的**最右侧**（即右上角）。
- **左边距与排版**：标题文字靠左对齐，关闭按钮靠最右侧对齐，二者水平拉开，绝对禁止将关闭按钮放置在标题文字的左侧。
- **排版实现原则**：标题栏整体必须采用水平 flex 布局（确保标题文字与关闭按钮自动分布在容器两端，左标题右按钮）。
- **Ant Design Vue 4.x 专项适配指南**（防范关闭按钮错位及高度异常）：
  - **关闭按钮右置**：在 Ant Design Vue 4.x 中，默认的 DOM 树顺序是关闭按钮（`.ant-drawer-close`）在标题（`.ant-drawer-title`）之前。要实现左标题右按钮，**必须**在 `.ant-drawer-header-title` 上使用 `flex-direction: row-reverse !important;`，或者通过 `order` 属性改变渲染顺序（标题 `order: 1`，关闭按钮 `order: 2`）。
  - **高度精准锁定**：AntD 默认对 `.ant-drawer-header` 设置了 `min-height: 56px`。要锁定 48px 高度，除了设置 `height: 48px !important;` 之外，**必须**显式设置 `min-height: 48px !important;` 清除其默认拉伸。

## 内容区域

- 左右边距为 32px
- 上下无额外间距，内容区域紧贴标题栏底部和底部按钮栏顶部
- 上下不得增加额外的 padding 或 margin
- 内容超出时，内容区域内部滚动
- 表单、表格、详情内容都必须适配内容区域
- 内容滚动不能影响标题栏和底部按钮栏

## 底部按钮栏

- 高度固定为 56px
- 有 1px 顶部分割线（无底色，使用 `border-top: 1px solid var(--color-graphite-l30)`）
- 按钮左对齐，主按钮 margin-left 为 16px（容器 padding-left 为 16px，合计距抽屉边缘 32px，确保与内容区对齐）
- 按钮高度：32px
- 按钮文字：14px
- 主按钮位于最左侧
- 取消按钮位于主按钮右侧

## 验收标准

- 抽屉默认从右侧滑出
- 抽屉宽度符合允许尺寸列表
- 标题栏和底部按钮栏固定不滚动
- 内容区域独立滚动
- 底部按钮栏始终可见
