# 下拉菜单组件 (Dropdown)

## 使用声明

本规范只定义组件的视觉结构、布局关系、交互行为和验收标准，不提供具体框架代码实现。

生成代码时，必须以当前项目的技术栈 Skill 为准。


## 使用场景

下拉菜单用于展开更多操作按钮，典型场景包括：

- 点击按钮后展开其他按钮操作（如"更多"操作）
- 点击可变更标签的下拉 icon 展开显示其他操作
- 右键菜单

下拉菜单不是选择器，不用于从选项中选取值。选取值的场景应使用 Select 选择器。

## 下拉面板规范

- 边框：无边框
- 圆角：2px
- 投影：var(--seer-dropshadow-s1)
- 上下内间距：4px
- 最小宽度：72px
- 宽度：根据操作文字长度自适应
- 高度：根据操作项数量自适应

## 菜单项规范

- 文字字号：12px
- 文字字重：400
- 文字颜色：var(--color-graphite-d40)
- 文字高度：24px（单行菜单项高度）
- 文字对齐：左对齐
- 文字左侧间距：12px
- 文字右侧间距：12px

## 交互状态

### Hover

- 背景色：var(--color-graphite-l50)
- 文字颜色：var(--color-graphite-d40)

### 禁用

- 文字颜色：var(--color-graphite)
- 光标：not-allowed
- Hover 背景不变

## 分隔线与分组

- 当菜单项需要分组时，组与组之间使用分隔线
- 分隔线颜色：var(--color-graphite-l30)
- 分隔线上下间距：4px
- 分组文字：12px，辅助色 var(--color-graphite)，文字高度 20px
- 分组文字与菜单项文字左对齐（左侧间距 12px）

## 验收标准

- 下拉面板无边框
- 圆角为 2px
- 投影为 var(--seer-dropshadow-s1)
- 菜单项文字 12px、正文色、左对齐、左侧间距 12px
- 菜单项高度 24px
- 面板上下内间距 4px
- 最小宽度 72px
- Hover 背景色正确
- 禁用项视觉可区分
