# 全局提示组件 (Message)

## 使用声明

本规范只定义组件的视觉结构、布局关系、交互行为和验收标准，不提供具体框架代码实现。

生成代码时，必须以当前项目的技术栈 Skill 为准。

如果本规范与项目技术栈约束发生冲突，优先遵守项目技术栈约束。

本规范禁止包含：
- React / Vue / Angular 代码示例
- JSX / TSX
- TypeScript 示例
- import 语句
- className
- React state 写法
- 具体图标组件名
- 具体组件库 API

本规范中的组件名称仅表示设计语义，不代表必须使用某个框架或组件库的同名组件。

本组件遵循 [全局原子规范与间距规范](design-atomic-spacing.md) 中的设计原则。

## 组件概述

用于不中断用户操作的轻量级反馈，通常显示在页面顶部中央，并在几秒后自动消失。

## 视觉规范

### 圆角
- 强制应用 2px 圆角

### 字体规范
| 规范项 | 规范值 |
|--------|--------|
| 内容文字字号 | 12px |
| 内容文字字重 | 400 |
| 内容文字行高 | 20px |
| 图标尺寸 | 16px |
| 图标左间距 | 16px |
| 图标与文字间距 | 8px |
| 内容文字右间距 | 12px |

### 物理位置
- 默认距离窗口顶部 15% 的高度处

### 展示限制
- 同一时间最多展示 5 条提示，超出时自动滚动或覆盖

### 长度限制
- 长度跟随提示文字长度自适应

### 阴影规范
- 无边框
- 使用 var(--dropshadow-s3): X0 Y8 B22 S0, 颜色 #1E232B 透明度 12%（见 [色彩体系-阴影规范](design-color.md#15-阴影规范)）

## 消息类型视觉规范

所有类型的背景色统一为白色，无边框，通过图标颜色区分等级。

| 类型 | 图标语义 | 图标/文字颜色 | 背景色 |
|------|---------|-------------|--------|
| 成功 | 成功语义图标（圆形勾选） | var(--color-turquoise-d10) | var(--color-white) |
| 信息 | 信息语义图标（圆形感叹号） | var(--color-blue) | var(--color-white) |
| 警告 | 告警语义图标（三角形感叹号） | var(--color-orange) | var(--color-white) |
| 错误 | 错误语义图标（圆形叉号） | var(--color-red-d10) | var(--color-white) |
| 加载 | 加载语义图标（旋转圆环） | var(--color-blue) | var(--color-white) |

## 交互规则

### 基础反馈
- 支持成功、信息、警告、错误、加载五种类型
- 默认 3 秒后自动消失
- 设置为 0 时不自动关闭，需手动关闭

### 手动关闭
- 加载提示需要手动关闭
- 支持自定义展示时长
- 关闭时可触发回调

### 自定义内容
- 支持自定义提示内容，包括带链接的提示
- 支持带关闭按钮的提示

## 注意事项

- 圆角必须为 2px
- 内容文字字号 12px，字重 400，行高 20px
- 默认距离窗口顶部 15% 的高度处
- 同一时间最多展示 5 条提示
- 长度跟随提示文字长度自适应
- 无边框，通过外阴影制造层次感
- 禁止使用非规范的颜色值

## 验收标准
- 消息提示圆角为 2px
- 文字字号和行高正确
- 消息位置在窗口顶部 15% 处
- 各类型颜色正确
- 自动消失时间正确
- 最多同时展示 5 条
