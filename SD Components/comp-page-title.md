# 页面标题/页头组件

## 使用声明

本规范只定义组件的视觉结构、布局关系、交互行为和验收标准，不提供具体框架代码实现。

生成代码时，必须以当前项目的技术栈 Skill 为准。

如果本规范与项目技术栈约束发生冲突，优先遵守项目技术栈约束。

本规范禁止包含：
- React / Vue / Angular 代码示例
- JSX / TSX
- TypeScript 示例
- import 语句
- className
- React state 写法
- 具体图标组件名
- 具体组件库 API

本规范中的 Header、Tab、Icon、Divider 等名称仅表示设计语义，不代表必须使用某个框架或组件库的同名组件。

## 目录

## 基础标题栏

结构：`[页面标题]`

适用场景：
- 一级页面标题
- 独立功能模块标题
- 不需要返回操作的页面

视觉规范：
- 高度：48px
- 字号：16px
- 字重：600
- 文字颜色：var(--color-graphite-d40)
- 背景色：var(--color-white)
- 垂直居中
- 无边框
- 左边距：16px，相对于内容卡片；或 12px，相对于内容卡片

验收标准：
- 标题文字垂直居中
- 标题栏高度稳定为 48px
- 标题栏不出现额外边框

## 下钻页标题栏

结构：`[返回入口] [页面标题]`

适用场景：
- 列表页进入详情页
- 多级页面导航
- 需要返回上一级的页面

视觉规范：
- 高度：48px
- 返回入口位于标题文字左侧
- 返回入口与标题文字间距：8px，特殊场景可为 12px
- 返回入口颜色：var(--color-graphite-d40)
- 返回入口 hover 颜色：var(--color-blue)
- 标题文字字号：16px
- 标题文字字重：600
- 标题文字颜色：var(--color-graphite-d40)

交互规则：
- 点击返回入口后，回到上一级页面或触发当前页面定义的返回逻辑
- 返回入口 hover 时需要有可点击反馈
- 返回入口不能影响标题文字对齐

验收标准：
- 返回入口和标题文字垂直居中
- 返回入口 hover 颜色正确
- 点击返回入口能触发返回行为

## 分层 tabs 页标题

结构：`[页面标题] [分割线] [Tab1] [Tab2]`

适用场景：
- 同一功能模块下存在多个视图
- 页面标题不变，但内容区需要通过 Tab 切换

视觉规范：
- 标题栏高度：48px
- 标题左边距：16px，相对于内容卡片
- 标题字号：16px
- 标题字重：600
- 标题颜色：var(--color-graphite-d40)
- 分割线宽度：1px
- 分割线高度：16px
- 分割线颜色：var(--color-graphite-l30)
- 标题与分割线间距：12px
- 分割线与 tabs 间距：12px
- Tab 字号：14px
- Tab 字重：400
- Tab 未选中颜色：var(--color-graphite-d30)
- Tab 选中颜色：var(--color-blue)
- 选中态下划线颜色：var(--color-blue)

下划线对齐规则：
- 选中态下划线必须与标题栏底部分割线在同一水平线上
- 需要避免出现双底线
- Tab 下划线应与标题栏底线形成连续视觉关系

交互规则：
- 点击 Tab 后切换当前视图
- 当前 Tab 进入选中态
- 其他 Tab 退出选中态
- Tab 切换不应导致标题栏高度变化

验收标准：
- 标题、分割线、Tab 垂直居中
- Tab 选中态清晰
- 下划线与标题栏底线对齐
- 切换 Tab 时标题栏不抖动

## 标题栏分割线规则

标题栏是否展示底部分割线，由标题栏所在页面容器决定，不作为标题栏结构类型区分。

分割线样式：
- 高度：1px
- 颜色：var(--color-graphite-l30)（一级分割线颜色）

### 在内容卡片内部使用时

- 可展示底部分割线
- 分割线跟随内容卡片左右内边距
- 如果内容卡片左右 padding 为 12px，则分割线左右也缩进 12px
- 分割线不通栏展示
- 分割线不额外占用高度，包含在标题栏 48px 内

### 在页面顶部独立模块使用时

- 可展示底部分割线
- 分割线通栏展示
- 分割线宽度等于标题栏所在模块宽度
- 分割线不额外占用高度，包含在标题栏 48px 内

### 不需要区分上下内容层级时

- 不展示分割线

### 实现约束

- 分割线不应通过额外 Divider DOM 实现
- 推荐使用标题栏自身 border-bottom 或 ::after 伪元素实现
- 避免 Divider 默认 margin 影响标题栏高度和内容间距
