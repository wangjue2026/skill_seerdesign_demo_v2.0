# 选择器组件 (Select)

## 使用声明

本规范只定义组件的视觉结构、布局关系、交互行为和验收标准，不提供具体框架代码实现。

生成代码时，必须以当前项目的技术栈 Skill 为准。

如果本规范与项目技术栈约束发生冲突，优先遵守项目技术栈约束。

本规范禁止包含：
- React / Vue / Angular 代码示例
- JSX / TSX
- TypeScript 示例
- import 语句
- className
- React state 写法
- 具体图标组件名
- 具体组件库 API

本规范中的组件名称仅表示设计语义，不代表必须使用某个框架或组件库的同名组件。

本组件遵循 [全局原子规范与间距规范](design-atomic-spacing.md) 中的设计原则。

## 未选择状态

### 视觉规范
- 高度：32px
- 圆角：2px
- 边框色：var(--color-graphite-l20)
- 背景色：var(--color-white)

### 提示文字规范
- 字号：12px
- 颜色：var(--color-graphite) #A1A7B3
- 文字左边距：12px
- 垂直居中对齐

### 下拉图标规范
- 右对齐
- 右间距：12px
- 垂直居中对齐

## 已选状态

### 单选选择器已选效果
- 文字颜色：var(--color-graphite-d40) #2F3540
- 边框色：var(--color-graphite-l20)
- 背景色：var(--color-white)
- 其他样式与未选择状态保持一致

### 多选选择器已选效果

多选选择器使用多个灰色标签呈现已选项。

#### 标签样式
- 标签文字颜色：var(--color-graphite-d40) #2F3540
- 标签背景：var(--color-graphite-l40) #EDF1F7
- 标签圆角：2px
- 标签高度：20px
- 标签之间间距：2px
- 标签左右内间距：8px

#### 关闭入口
- 尺寸：12px x 12px
- 关闭入口与文字间距：8px

## 下拉面板规范
- 圆角：2px
- 选项高度：32px
- 选中态背景色：var(--color-blue-l50)，底色通栏展示
- 选中态文字颜色：var(--color-blue)
- Hover 背景色：var(--color-graphite-l50)
- 面板上下间距：4px
- 面板阴影：var(--dropshadow-s2): X0 Y4 B16 S0, 颜色 #1E232B 透明度 14%（见 [色彩体系-阴影规范](design-color.md#15-阴影规范)）
- 禁止勾选图标：选中项不显示勾选图标

## 激活/选中状态 (Focus)
- 边框色：var(--color-blue)
- 焦点阴影：严禁使用蓝色外发光阴影

## 禁用状态
- 背景色：var(--color-graphite-l30)
- 边框色：var(--color-graphite-l20)
- 文字颜色：var(--color-graphite)
- 光标：not-allowed

## 多选选择器已选项溢出规则

多选选择器在表单中默认使用单行折叠展示。

规则：
- 选择器高度固定为 32px
- 已选标签高度为 20px
- 已选标签不允许换行
- 已选标签不允许溢出选择器
- 当已选项无法完整展示时，剩余项折叠为 +N
- +N 数量根据选择器剩余宽度自动计算
- 不允许因为已选项过多自动撑高选择器
- 不允许已选标签覆盖右侧下拉箭头
- 多选选择器必须显式开启 showArrow
- 多选选择器必须使用响应式 maxTagCount
- 折叠标签 +N 样式与普通已选标签保持一致

## 验收标准
- 选择器圆角为 2px
- 未选择状态边框色正确，提示文字样式符合规范
- 下拉图标右对齐，右间距正确
- 提示文字和下拉图标垂直居中对齐
- 单选选择器已选文字颜色正确
- 多选选择器标签样式符合规范
- 标签高度、圆角、间距正确
- 关闭入口尺寸和间距正确
- 下拉面板选中项无勾选图标
- 选中底色通栏展示
- 激活态边框颜色正确
- 禁用状态视觉可区分
