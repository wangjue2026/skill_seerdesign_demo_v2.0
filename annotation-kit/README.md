# 交互自助标注工具 (Annotation Kit - Live Dev Agent 版)

## 目的

本工具套件提供一个 **全屏自适应、高级暗黑风格** 的页面标注方案，让您的 Vue 项目拥有一套极具科技感（Live Dev Agent 风格）的批注控制台。标注数据以百分比坐标系进行存储（完美兼容屏幕缩放），并以 **内联 JSON 注释** 的形式保存在对应 `.vue` 文件的末尾（`/*@annotations{...}@*/`），保证标注随代码一起提交、共享。

## 核心升级特性

1. **悬浮式控制台**：全新的右侧控制台采用悬浮设计，展开时绝对不会挤压或影响您原本的业务页面布局。
2. **快捷打点入口**：在右下角的悬浮胶囊上新增了快捷的「📍 打点」按钮，无需展开控制台即可随时发起位置标定。
3. **响应式坐标系**：打点坐标自动计算为 `window.innerWidth / window.innerHeight` 的百分比（`xPercent`, `yPercent`），确保在不同分辨率下标记位置始终精准。
4. **自闭环数据重写**：采用“复制最新源码”的方式导出数据，安全可靠，不依赖 Node.js 的实时写入。

## 结构

```
annotation-kit/
├─ annotation-tool.js      # Node 脚本，自动向目标 .vue 文件注入 AnnotationPanel 组件
├─ annotation-data.js      # 读取/写入 inline JSON 注释的工具函数（供辅助脚本使用）
├─ annotation-panel.vue    # Live Dev Agent 风格核心 UI 组件
└─ README.md               # 本说明文档
```

## 使用方式

### 1. 注入面板

在终端执行以下命令，将 AnnotationPanel 注入到您的目标页面中：

```bash
# 示例：为 SaseTerminalControl.vue 添加标注组件
node annotation-kit/annotation-tool.js /Users/wj/Desktop/项目文件/SD规范Demo/src/components/demos/SaseTerminalControl.vue
```

### 2. 进行标注与打点

注入成功并在浏览器刷新页面后，您会在右下角看到一个**悬浮胶囊控制条**：
- **快捷打点**：点击胶囊上的 `📍 打点` 按钮，屏幕会出现透明遮罩与十字光标。在目标位置点击即可弹出打点详情窗口，输入标题和内容。
- **展开控制台**：点击胶囊上的 `🛠️ 控制台` 按钮即可滑出右侧的高级批注面板，您可以在里面查看所有的交互说明、统一业务语言以及代码源码。
- **显示/隐藏点位**：通过胶囊上的 `说明` 开关，可以一键隐藏画布上的打点信息，还原最纯净的业务展示。

### 3. 持久化数据（重要！）

由于浏览器端无法直接向本地文件系统写入文件，新版的工具采用了**“剪贴板自闭环”**的数据持久化设计：
1. 当您完成所有打点和编辑后，打开右侧控制台，点击右上角的绿色按钮 **「📋 复制最新源码」**。
2. 此时，所有最新的打点数据已被格式化为 `/*@annotations{...}@*/` 注释块并存入您的系统剪贴板。
3. 回到您的 IDE，打开对应的 `.vue` 文件，将剪贴板的内容直接**粘贴到文件最底部**。
4. 保存文件，Git 提交即可永久固化批注数据！

## 样式说明

`annotation-panel.vue` 采用了纯净的 `<style scoped>` Vanilla CSS 编写，**100% 零外部依赖**。它不会依赖任何 Tailwind CSS，也不会污染您项目的全局样式。
- 采用高级暗色玻璃拟态（backdrop-filter）。
- 包含呼吸灯动效（Pulse Radar）。
- 高度还原现代网络安全/管理后台的专业质感。

## 常见问题

| 问题 | 解答 |
|------|------|
| 刷新页面后我刚打的点不见了？ | 在没有点击“复制最新源码”并粘贴到代码里之前，临时数据保存在浏览器的 `localStorage` 中。如果是硬刷新清理了缓存，请确保您已经将代码粘贴到了 `.vue` 文件底部。 |
| 打点时被原本页面的按钮遮挡？ | 工具的打点遮罩层 `z-index` 设置为 `999998`。如果您的页面中有极端高的层级（如 999999），请适当调整以保证打点遮罩层在最上方。 |
| 想撤销注入？ | 删除目标文件顶部的 `import AnnotationPanel ...`、 `<AnnotationPanel>` 标签、 `components` 注册，以及底部的注释块即可。 |
